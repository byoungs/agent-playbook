# Key Reading List for Agentic Development

> Curated 2026-03-27. Prioritized by signal-to-noise ratio.

## Must-Read (Tier 1)

### Anthropic Engineering
- [How We Built Our Multi-Agent Research System](https://www.anthropic.com/engineering/multi-agent-research-system) — The definitive orchestrator-worker pattern. Multi-agent outperforms single by 90%.
- [Effective Harnesses for Long-Running Agents](https://www.anthropic.com/engineering/effective-harnesses-for-long-running-agents) — Initializer + coding agent pattern. Progress files. Never delete tests.
- [Effective Context Engineering for AI Agents](https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents) — Context is the bottleneck, not intelligence.
- [Claude Code: Best Practices](https://www.anthropic.com/engineering/claude-code-best-practices) — Official guide from the team.
- [2026 Agentic Coding Trends Report](https://resources.anthropic.com/2026-agentic-coding-trends-report) — Survey of ~1,000 developers.

### Practitioner Workflows
- [Boris Cherny: howborisusesclaudecode.com](https://howborisusesclaudecode.com) — 57 tips from the Claude Code creator.
- [Harper Reed: My LLM Codegen Workflow](https://harper.blog/2025/02/16/my-llm-codegen-workflow-atm/) — Two-phase spec→plan workflow.
- [Addy Osmani: AI Coding Workflow](https://addyosmani.com/blog/ai-coding-workflow/) — Chunked workflows, accountable engineer.

### Methodology
- [Kent Beck: Augmented Coding Beyond the Vibes](https://tidyfirst.substack.com/p/augmented-coding-beyond-the-vibes) — TDD originator on AI agents.
- [Martin Fowler: Humans and Agents in Software Engineering Loops](https://martinfowler.com/articles/exploring-gen-ai/humans-and-agents.html) — "On the loop" framework.

## Highly Recommended (Tier 2)

### Quality Patterns
- [CodeScene: Best Practice Patterns for Speed with Quality](https://codescene.com/blog/agentic-ai-coding-best-practice-patterns-for-speed-with-quality) — 6 quality patterns with quantitative thresholds.
- [Qodo: AI Code Review Predictions 2026](https://www.qodo.ai/blog/5-ai-code-review-pattern-predictions-in-2026/) — Specialist-agent review pattern.

### Frameworks and Handbooks
- [Tweag Agentic Coding Handbook](https://tweag.github.io/agentic-coding-handbook/) — Controlled experiment showed 45% faster with AI. Spec-first + TDD.
- [Simon Willison: Agentic Engineering Patterns](https://simonwillison.net/2026/Feb/23/agentic-engineering-patterns/) — Weekly chapters, honest practitioner perspective.
- [GitHub: Spec-Driven Development Toolkit](https://github.blog/ai-and-ml/generative-ai/spec-driven-development-with-ai-get-started-with-ai/) — Spec as contract for agents.

### Industry Perspective
- [Mike Mason: AI Coding Agents 2026](https://mikemason.ca/writing/ai-coding-agents-jan-2026/) — Sobering data: DORA, METR, LinearB studies.
- [Steve Yegge: Six New Tips](https://steve-yegge.medium.com/six-new-tips-for-better-coding-with-agents-d4e9c86e42a9) — Rule of Five, Merge Wall.

### Podcasts / Interviews
- [Lenny's Podcast: Boris Cherny](https://www.lennysnewsletter.com/p/head-of-claude-code-what-happens) — "What happens after coding is solved"
- [YC Lightcone: Inside Claude Code](https://www.ycombinator.com/library/NJ-inside-claude-code-with-its-creator-boris-cherny)
- [Pragmatic Engineer: Kent Beck on TDD + AI](https://newsletter.pragmaticengineer.com/p/tdd-ai-agents-and-coding-with-kent)
- [Latent Space: AI Agents Meet TDD](https://www.latent.space/p/anita-tdd)

## Good Background (Tier 3)

- [OpenAI Codex Workflows](https://developers.openai.com/codex/workflows) — PM→Designer→Dev→Tester pattern
- [GitHub Agentic Workflows](https://github.github.com/gh-aw/) — "Continuous AI" in Actions
- [Latent Space: Agent Engineering](https://www.latent.space/p/agent) — LLM-driven control flow
- [HuggingFace: Agentic Coding Trends Implementation Guide](https://huggingface.co/blog/Svngoku/agentic-coding-trends-2026)
